# multi-player-word-game
Para jugar entre 1 a 4 jugadores desde una misma PC. El juego consiste en ir escribiendo palabras que cumplan con los requisitos aleatorios que le toque a cada jugador. Hay un limite de tiempo y no se permite repetir palabras que hayas escrito vos u otro jugador. Si no cumples con los requerimientos no puedes seguir jugando.
